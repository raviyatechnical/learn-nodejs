## Run Locally

Clone the project

```bash
  git clone https://github.com/raviyatechnical/learn-nodejs
```

Go to the project directory

```bash
  cd learn-nodejs
```

Copy .env.example to .env
```bash
  cp .env.example .env
```

Install dependencies

```bash
  npm install
```

Start the server for development

```bash
  npm run dev
```

## Run Server 

Start the server

```bash
  npm run start
```
